# 朝ハムバンク PWA

iPhoneのホーム画面に置けるミニWebアプリです。

## 使い方

1. このフォルダの中身を GitHub Pages などに置く
2. iPhoneのSafariで公開URLを開く
3. 共有ボタン → 「ホーム画面に追加」
4. ホーム画面の「朝ハム」アイコンから開く

## 注意

ChatGPTアプリ内や sandbox リンクのプレビューでは、スタンプ・ごほうび追加・保存が正常に動かないことがあります。
Safariで公開URLとして開くと動作確認しやすいです。

## 内容

- index.html: 本体
- manifest.webmanifest: PWA設定
- sw.js: オフライン用キャッシュ
- icons/: ホーム画面アイコン
